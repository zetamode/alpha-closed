La version Alpha est fermée

Le 28 août, après 888 jours d’existence quasi non-stop, la version Alpha a été arrêtée "définitivement".

Nous avons pris cette décision afin de pouvoir nous concentrer pleinement sur la campagne de financement qui se profile à l'horizon. (Vous ne voyez pas cette ombre qui s'agrandit ?). Cela allégera également nos frais actuels. ZetaMode reverra le jour après la campagne (espérons en version Beta cette fois !).

Afin de remplacer le Forum Alpha durant ce temps, nous avons mis en place un Live Chat à l'adresse https://chat.zetamode.com/ (la partie HTTPS est très importante !)

Si vous n'avez plus visité l'Alpha depuis quelques semaines, vous n'avez pas encore vos identifiants pour le Live Chat. Écrivez-nous un mail et on vous donnera l'accès fissa. Idem si vous avez un problème avec le login, contactez-nous !

Gardez également un œil sur notre page d'accueil et sur les réseaux sociaux (Twitter, Facebook) pour la suite des opérations !
